DECLARACION IMPLICITA DE VARIABLES:
    var edadPersona = 27 -->  se le dice al compilador que asigna al tiempo de ejecucion el tipo que corresponda a esa variable. En este caso edadPersona se le asignara el tipo int ya que se le asigno a la variable un valor entero (27)

CONVERSION EXPLICITA(casting)
    double tempe= 34.5;
    int tempeMadrid;

    temperatura = (int) temperatura;  ---> basicamente dice que al valor double que es tempe lo pase a entero y se lo asigne a tempeMadrid. (desecha la parte decimal, NO REDONDEA)

    int.parse()
    double.parse()

GIT/GITHUB
------------
git config -h --> listado de configuraciones

git init 	--> inicializar repositorio nuevo (Vacio)
ls -a 		--> mostrar todos los archivos incluso los ocultos
cd .git 	--> ingresar al archivo .git
git add <file>--> sube archivo al stage

git commit -m "first commit" --> subir archivos al repositorio local

git rm <file> 	-->eliminar archivo

git restore <file> --> restaurar archivo

git branch -> rama actual

git checkout -b <nombreRama> --> cambiar de rama

git log --> Historial

cat <file>  ---> muestra contenido del archivo

git push -u origin <rama> --> subir a github el archivo
